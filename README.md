# Tribute Page for Dr. A. P. J. Abdul Kalam

This project is a simple and elegant tribute page created using HTML and CSS. It is dedicated to Dr. A. P. J. Abdul Kalam, one of India’s most admired scientists, teachers, and leaders.

The page highlights his inspiring life, achievements, vision, famous books, awards, and memorable quote. It also includes images, a responsive layout, attractive typography, and a visually appealing color theme.

## Features

- Clean and responsive tribute page design
- Dedicated sections for admiration, qualities, and achievements
- Highlight panel with major milestones from Dr. Kalam’s life
- Image gallery for a richer presentation
- Stylish fonts, background effects, and modern card layout
- Built using only HTML and CSS

## Technologies Used

- HTML5
- CSS3
- Google Fonts

## Project Purpose

The purpose of this project is to practice basic web development skills by creating a meaningful and visually appealing webpage for a person I admire. This project shows how HTML and CSS can be used to build a beautiful tribute page with proper structure, images, text formatting, and design.

## Tribute Subject

Dr. A. P. J. Abdul Kalam was chosen for this tribute because of his humility, intelligence, service to the nation, love for students, and inspiring vision for India.

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html` to open it in your browser.
